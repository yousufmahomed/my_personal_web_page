Implementing a Least Privilege Security Model in Salesforce

Project Overview

This project demonstrates the architectural design and deployment of a custom Salesforce CRM focused on sales pipeline coordination. Rather than relying on out-of-the-box settings, the objective was to design a highly secure, scalable environment utilizing a "Least Privilege" data access model for a multi-tiered sales team consisting of Sales Managers, Sales Closers, and Appointment Setters.

The Architectural Thinking

"The core challenge of any enterprise CRM is balancing workflow efficiency with data security. This project proves that you can protect sensitive financial data while still providing a frictionless user interface for top-of-funnel staff."

1. Baseline Security & Role Hierarchy

To ensure robust data protection, the baseline Organization-Wide Defaults (OWD) for Leads and Opportunities were locked down to Private. Access was then systematically opened vertically using a custom Role Hierarchy. This ensures that Sales Managers automatically inherit visibility into all subordinate records without requiring complex sharing rules.

2. Permission-Set-Led Field-Level Security (FLS)

A modern Salesforce architecture relies less on profile-cloning and more on flexible permission sets. To achieve this:

  • Created a unified Custom: Base Sales User profile for all frontline staff.
  • Built custom Opportunity fields for sensitive data (Discount Amount, Incentive Details, Contract Terms, Final Quotation Price).
  • Applied strict FLS during field creation to render these fields invisible to the Base Profile.
  • Engineered a Sales Closer Privileges Permission Set to explicitly grant read/edit access to these financial fields only to authorized Closers and Management.

3. Process Governance (Validation Rules)

To prevent top-of-funnel users from bypassing standard operating procedures, I implemented a role-based Validation Rule. Appointment Setters are physically blocked by the system from moving a Lead to a "Closed" status, ensuring all hand-offs go through a Sales Closer for final review.
